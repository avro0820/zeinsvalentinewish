
# Valentine's Day Message

This project is an interactive Valentine's Day message with floating hearts, a personalized message, and a button that shows sweet messages when clicked. It is built with HTML, CSS, and JavaScript.

## Features

- Interactive floating hearts animation.
- Sweet and personalized Valentine's Day messages.
- Clickable button to reveal more messages.

## Project Structure

```
.
├── index.html           # The main HTML file for the webpage
├── style.css            # Styles for the page (optional, can be inline)
├── script.js            # JavaScript code for interactive features
└── assets/              # Folder for images or other media (optional)
```

## How to Use

1. Clone the repository or download the files.
2. Open the `index.html` file in a web browser.
3. Enjoy the Valentine's Day message and the floating hearts effect!
